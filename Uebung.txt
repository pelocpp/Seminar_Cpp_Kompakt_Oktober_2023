Teuflische Folge

Startwert n;

// int n = 7;

Nächstes Folgenglied:

n gerade:   ==> n / 2

n ungerade: ==> 3 * n + 1

Bei n == 1 Ende der Folge erreicht.

Beispiel:

7, 22, 11, 34, 17, 52, 26, 13, 40, 20, 10, 5, 16, 8, 4, 2, 1.

Bis heute konnte mathematisch NICHT bewiesen werden,
dass diese Folge für JEDEN Startwert nach endlich vielen Schritten terminiert.
